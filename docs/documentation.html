1. Introduction

CampusIQ is an enterprise-grade mobile application designed exclusively for college and university administrators. Built with React Native, it provides a unified platform for managing institutional operations, tracking compliance, and making data-driven decisions.

Key Differentiators:

  • Admin-only platform - no student-facing features
  • AI-powered task analysis using Gemini
  • Real-time synchronization with Firebase
  • Offline-first architecture
  • Executive dashboard with performance metrics

2. Target Users

CampusIQ is designed for authorized institutional personnel only:

Role Access Level
Principal / President Full executive access
Dean Department-level operations
Registrar Academic records & compliance
Director Functional area management
Management Committee Strategic oversight

3. Setup and Installation

Prerequisites

  • Node.js 18+ (recommended: 20.x)
  • React Native CLI
  • Android Studio (for Android development)
  • Xcode (for iOS development)

Installation Steps

# Navigate to project directory
cd crm-lead-management-app-react-native

# Install dependencies
npm install

# Start Metro bundler
npm start

# Run on Android
npm run android

# Run on iOS (macOS only)
npm run ios

6. Firebase Configuration

  1. Create a Firebase project at console.firebase.google.com
  2. Enable Authentication (Email/Password)
  3. Enable Firestore Database
  4. Enable Cloud Messaging
  5. Download and add configuration files:
    • google-services.json → /android/app/
    • GoogleService-Info.plist → /ios/CRM/

7. Environment Variables

Create a .env file in the project root:

GEMINI_API_KEY=your_gemini_api_key
FCM_SERVER_KEY=your_fcm_server_key

9. Security

  • Authentication: Firebase Auth with institutional email
  • Authorization: Role-based access (ADMIN only)
  • Data Privacy: Firestore security rules
  • Transport: HTTPS/TLS encryption

10. Troubleshooting

Build Issues

# Clear cache and reinstall
rm -rf node_modules
npm install
npm start --reset-cache

Android Issues

cd android
./gradlew clean
cd ..
npm run android
