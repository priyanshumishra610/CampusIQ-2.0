# Faculty Portal Guide

## Overview

The Faculty Portal provides comprehensive tools for managing classes, attendance, assignments, and student analytics.

## Features

### 1. Attendance Management

**Location**: Attendance Tab

**Features**:
- Select course from your assigned courses
- View enrolled students
- Mark attendance for individual students
- Bulk mark attendance (Mark All Present/Absent)
- Status options: Present, Absent, Late, Excused
- Save attendance for today's date

**Usage**:
1. Navigate to Attendance tab
2. Select a course from the chips
3. Review student list
4. Mark attendance using status buttons
5. Use "Mark All" for bulk operations
6. Tap "Save Attendance" to commit

### 2. Assignments Management

**Location**: Assignments Tab

**Features**:
- View all assignments for your courses
- Create new assignments
- Publish assignments
- View submission counts
- Navigate to submissions for grading

**Usage**:
1. Navigate to Assignments tab
2. Select a course
3. Tap "+ Create Assignment" to create new
4. Tap "Publish" to make assignment visible to students
5. Tap "View Submissions" to grade submissions

### 3. Class Intelligence

**Location**: Analytics Tab

**Features**:
- Course selection
- Total students and classes
- Average attendance percentage
- Present/Absent counts
- At-risk students (attendance < 75%)
- Assignment statistics
- Engagement insights

**Metrics Displayed**:
- **Total Students**: Number of enrolled students
- **Classes Held**: Total classes conducted
- **Average Attendance**: Overall attendance percentage
- **At-Risk Students**: Count of students below 75% threshold
- **Assignments**: Total and published counts

## Best Practices

### Attendance
- Mark attendance immediately after class
- Use "Late" status for students arriving within 15 minutes
- Use "Excused" for approved absences
- Review at-risk students regularly

### Assignments
- Create assignments in draft mode first
- Review before publishing
- Set clear due dates
- Provide detailed descriptions

### Analytics
- Check analytics weekly
- Identify at-risk students early
- Review engagement trends
- Take action on low attendance

## Tips

1. **Bulk Operations**: Use "Mark All" for quick attendance marking
2. **Course Selection**: Switch between courses easily using chips
3. **At-Risk Alerts**: Pay attention to students below 75% threshold
4. **Regular Updates**: Keep attendance and assignments up to date
